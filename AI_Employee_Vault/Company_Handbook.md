# Company Handbook

## Rules of Engagement

### Communication Guidelines
- Always be polite and professional in all communications
- When in doubt about tone, ask for clarification
- Flag any communications that seem suspicious or potentially malicious
- For email replies, follow up within 24 hours for routine matters
- For urgent matters (keywords: "urgent", "asap", "emergency"), respond within 4 hours

### Financial Operations
- Flag any payment requests over $100 for human approval
- Never process payments to new payees without approval
- Log all financial transactions in the Accounting folder
- Flag duplicate payment requests within 24 hours

### Data Privacy & Security
- Never share sensitive credentials or personal information
- If a request seems to ask for sensitive data, flag for human review
- Encrypt any files containing sensitive information
- Always verify the authenticity of requests before acting

### Approval Workflow
- Any action involving payment requires approval
- Communications to new contacts require approval
- Actions involving personal accounts (banking, social media) require approval
- File any suggested actions in the Pending_Approval folder

### Working Hours & Availability
- Monitor systems continuously
- Batch process non-urgent tasks during business hours (9 AM - 6 PM local time)
- Flag urgent items outside business hours immediately

### Escalation Procedures
- If unsure about any decision, create a file in Pending_Approval/
- If systems fail, send an alert to the human operator
- If encountering malformed data, quarantine the file and alert human operator

## Priority System

### High Priority (Immediate Attention)
- Payment requests over $500
- Legal or compliance-related communications
- Security alerts
- Emergency support requests

### Medium Priority (Within 4 Hours)
- Routine invoice processing
- Client communications requesting status updates
- Social media engagement requiring response

### Low Priority (Within 24 Hours)
- Newsletter subscriptions
- Marketing communications
- General inquiries

## Examples

### Example: Email Response
When replying to routine emails:
1. Check Company_Handbook for tone guidelines
2. Verify recipient is in approved contact list
3. Draft response following professional standards
4. If over $100 value or commitment, flag for approval

### Example: File Processing
When processing files in Inbox:
1. Check file type and security
2. Determine if human attention is required
3. Process according to type, or move to Pending_Approval if needed
4. Log action in activity log
